# nikki-daily_MISSION

 ### 必要设置
没压力挖掘分配：
1. 任意植物
2. 任意昆虫
3. 发夹泥鳅
4. 随意

理由：
植物和昆虫可以顺带完成每日任务中的"5植物"和"3昆虫"。因为除了蜘蛛之外的昆虫会乱动，所以这个任务最好直接在挖掘里完成。

而鱼类的挖掘量相当大，每次可以获得12.5个单位的鱼，在完成"素材激化幻境"的每日任务时优势巨大

2. 视角俯仰辅助-关闭
